# React-ThreeJS boilerplate

## ThreeJS

- install: `npm install`
  - `npm install three --save`
  - if it is typescript project, `npm i --save-dev @types/three`
- cd webgl-study
- `npm run start`

## using ThreeJS

```javascript
import * as THREE from 'three';
//扩展库
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls.js';
```
